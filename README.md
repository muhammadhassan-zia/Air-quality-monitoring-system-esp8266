# Air Quality Monitoring System

## Overview
This project is an IoT-based Air Quality Monitoring System designed to measure environmental conditions in real time.

## Features
- Monitors temperature and humidity using DHT11
- Detects air quality and gas levels using MQ-135 sensor
- Displays data on LCD
- Supports wireless data transmission via ESP8266

## Components Used
- NodeMCU (ESP8266)
- MQ-135 Gas Sensor
- DHT11 Temperature & Humidity Sensor
- LCD Display
- Power Supply & Supporting Circuitry

## Working Principle
The sensors collect environmental data which is processed by the ESP8266 microcontroller. The system displays the data locally on an LCD and can transmit it wirelessly for remote monitoring.
